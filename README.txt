# 🎮 Top 8 Generator — Mode d'emploi

## Installation
1. Extrais le dossier ZIP où tu veux sur ton ordi
2. Double-clique sur **index.html** — ça s'ouvre dans ton navigateur

## Utilisation à chaque tournoi

### 1. Importer les résultats (start.gg)
- Entre ta **clé API start.gg** dans le champ prévu (elle reste dans ton navigateur)
- Colle le **lien du tournoi** start.gg
- Clique sur **"Importer les résultats"** — les noms se remplissent automatiquement

### 2. Paramètres
- Sélectionne le **jeu** du tournoi
- Vérifie le **nom** et le **numéro** du tournoi (remplis automatiquement depuis start.gg)

### 3. Fond
- Clique pour charger ton **image de fond**
- Tu peux la changer à chaque tournoi sans rien reconfigurer

### 4. Joueurs & Personnages
- Vérifie les **noms** importés (tu peux les corriger manuellement)
- Clique sur **"Choisir"** pour assigner un personnage à chaque joueur

### 5. Aperçu & Export
- Clique sur **"Générer l'aperçu"** pour voir le résultat
- Modifie le **message Twitter** si besoin
- Clique sur **"Télécharger l'image"** — l'image est exportée en 1400×1400px

## Obtenir une clé API start.gg
1. Va sur https://start.gg/admin/profile/developer
2. Connecte-toi à ton compte
3. Crée un nouveau token
4. Copie-le dans l'outil

⚠️ Ne partage jamais ta clé API !

## En cas de problème
- **"Failed to fetch"** → Vérifie ta connexion internet et ta clé API
- **Cases mal placées** → L'outil est calibré pour la template Smash Ultimate fournie
- **Noms non importés** → Vérifie que le lien contient bien /tournament/.../event/...
